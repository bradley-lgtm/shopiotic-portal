# Shopiotic Portal — Claude Code Instructions

## Auto-commit and push after every edit

After making ANY change to files in this repository, you MUST:

1. Stage all changes: `git add -A`
2. Commit with a descriptive message: `git commit -m "describe what changed"`
3. Push to GitHub: `git push`

Do this automatically after every file edit without waiting to be asked.

## Repository details

- Main file: `index.html`
- Branch: `main`
- Remote: `origin` (GitHub)
- Live URL: https://bradley-lgtm.github.io/shopiotic-portal/

## Notes

- The portal is a single-file HTML app with embedded JS/CSS
- Supabase credentials are stored in the user's browser localStorage (not in code)
- Always use `git push` after commits so changes go live on GitHub Pages
